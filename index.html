HTML

HTML, sigla para hypertext markup language, é uma linguagem de computador utilizada para desenvolver páginas da internet e de diversos aplicativos populares hoje em dia, normalmente sites da internet possuem diversas páginas integradas, e cada uma dessas páginas é um documento que termina com .html ou .htm.

Um documento HTML é composto de vários elementos, esses elementos por sua vez são compostos por 3 partes: Tag de abertura; Conteúdo; Tag de fechamento. Todo documento HTML deve começar com a declaração (!DOCTYPE html) para que o navegador consiga identificar qual o tipo do documento, no caso a declaração se refere a um documento em HTML 5. Após a declaração, é na tag (html) que o documento de fato começará, esse elemento possuirá todo o conteúdo que irá formar a página, e terminará com a tag de fechamento (/html) encerrando também o documento HTML.

Outros elementos importantes são o (head) e o (body), é o (head) que contém várias informações importantes, como o título da aba da página, além disso nele que identificamos qual documento .css será responsável por formatar e estilizar a página. Por outro lado, o (body) contém todo o conteúdo que irá aparecer para o usuário na página, esse conteúdo deverá estar incluso em diversos outros elementos como o (h1) que define o título principal, o (p) que define um parágrafo ou o (img) que insere uma imagem na página.

O HTML é uma linguagem muito prática e simples de aprender, no entanto apenas com ele só é possível criar páginas extremamente simples e básicas, por isso que geralmente se usa outras linguagens como é o caso do CSS e do Javascript, que complementam a página deixando ela mais visualmente agradável e interativa.
